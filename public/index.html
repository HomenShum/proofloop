when it’s done.

Coding agents write code and say done. Proof Loop is the supervisor that plans the target, runs a gate against your app, and refuses false completion — bring any agent, one prompt starts the loop.

zsh

$ npx proofloop target --write-runner-plan

$ npx proofloop gate

  1. 01

    The gate reads receipts, not self-reports.

    Default check-only mode reads .proofloop/gate-state.json. A failing gate blocks stop and prints the failing checks — no subprocess, no network call.

  2. 02

    A forged pass is blocked, not trusted.

    The PreToolUse guard refuses edits to .proofloop/, proofloop.config.json, and .github/workflows/, and scans attempted edits for verifier-weakening patterns.

  3. 03

    Zero deps. Any repo. Install-free.

    MIT licensed, Node ≥ 20, zero runtime dependencies. npx proofloop works without adding anything to package.json.

proofloop.live keeps the same honesty boundary as the CLI: product-path proof, proxy benchmark proof, and official scorer output are always labeled separately — never blurred into one claim. The CLI can now run npx proofloop target --url ... to recommend benchmark families and record missing adapters. Want Proof Loop run against your app without setting up the harness yourself?
