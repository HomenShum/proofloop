# collaboration.md

ProofLoop collaboration roles:

- User: owns goal, permissions, budget, and final product judgment.
- Coding agent: changes code and fixes failures.
- ProofLoop: plans checks, supervises execution, protects receipts, and reports blockers.
- CI: clean-room backstop for gate commands.
- Hosted service: optional runner for teams that cannot run the harness locally.

No role should grade its own unverified output.
