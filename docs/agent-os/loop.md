# loop.md

ProofLoop runs a verification loop:

1. Observe the repo and optional live URL.
2. Build or refresh the target plan.
3. Identify benchmark families, scripts, gates, UI contracts, and blockers.
4. Generate or reuse runner tasks.
5. Run deterministic gates or supervised runner tasks.
6. Verify receipts.
7. Write charts, reports, and next actions.
8. Keep unresolved blockers visible.

The loop is allowed to continue work. It is not allowed to mark its own work done without proof.
